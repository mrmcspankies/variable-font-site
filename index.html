History

Science gothic is a libre sans variable font commissioned by Google fonts.The Font was offically released on December 21st 2022 according to Github with ownership of the font transferring to google. It was created by Thomas Phinney, Vassil Kateliev, Brandon Buerkle. This variable font with 4 axes is equivalent to 200+ different fonts. This font's medium weight is based on bank gothic which was created in 1930-1934 by Morris Fuller Benton for American type founders. American Type Founders is a business trust created in 1892 which merged 23 type foundries which was 85% of type created in the united states in the early 20th century.
